Author : Baptiste Degryse

This game is made in Oz langage, a multi-paradigm langage developed by a teacher in UCL (Peter van roy). It will not work unless you install it.

Write “make play” in the terminal in the game folder to launch it.

The keys for Player 1 are:
(Wind player)
Move
 c,b,g,v respectively left, right, up, down

Attack
 a,z,e,r then a move key

The keys for Player 2 are:
(Earth player)
Move
 The arrow keys (left, right, up down)

Attack
 j,k,l,m then a move key
